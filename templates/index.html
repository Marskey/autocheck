    Help

    本工具为定时自动检查指定目录中代码用于找出bug和安全漏洞.

    提示 本工具利用的是第三方的检查工具例如PVS-Studio检查结果与本人无瓜!

    上方导航栏中的本地项目目录中请填写使用者本机的对应项目的根目录地址（.sln所存在目录）当点击下载原报告文件时用来替换原报告文件的代码地址方便定位

    左边的表格内容为当前版本号对应报告分析，表格从左到右从上到下依次为

    File
    检查器查出有误的文件名
    Project
    项目名字
    Time
    检查器完成检查的时间
    Result
    检查器给出的报告
    该文件在检查版本范围内的提交日志

    本工具定时9点， 12点， 15点， 18点， 21点的时候开始检查, 检查区间为配置文件中的起始版本到最新的版本

    也可以直接点击 开始检查 按钮来立即执行检查流程, 区间为按钮旁边所填的区间范围，填Head表示到最新的一次提交

    提示 无论是自动检查还是通过点击 开始检查 按钮都会执行所有的检查器进行检查
